# Cullen and Frey Plot **\[deprecated\]**

Plots a Cullen and Frey graph of the skewness and kurtosis for
non-censored data.

## Usage

``` r
ssd_plot_cf(data, left = "Conc")
```

## Arguments

- data:

  A data frame.

- left:

  A string of the column in data with the concentrations.
