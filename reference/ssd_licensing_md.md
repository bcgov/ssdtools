# Licensing Markdown

A string of markdown code indicating the licensing of the code and
documentation

## Usage

``` r
ssd_licensing_md()
```

## Examples

``` r
ssd_licensing_md()
#> [1] "## Licensing\n  Copyright 2015-2023 Province of British Columbia\\\n  Copyright 2021 Environment and Climate Change Canada\\\n  Copyright 2023-2025 Australian Government Department of Climate Change, Energy, the Environment and Water\n\n  The documentation is released under the\n  [CC BY 4.0 License](https://creativecommons.org/licenses/by/4.0/)\n\n  The code is released under the\n  [Apache License 2.0](https://www.apache.org/licenses/LICENSE-2.0)\n"
```
